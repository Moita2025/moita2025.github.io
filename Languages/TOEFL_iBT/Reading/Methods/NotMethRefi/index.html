    阅读笔记法 + 精读标准

    笔记三大目的

    • 代替纸笔阅读中的勾画
    • 深入理解文章逻辑框架
    • 形成信息索引解题依据

    笔记内容

    • 要点信息：文章的结构主干（主题句、分论点）
    • 重点信息：支撑性事实、重点细节

    经典笔记符号

    • 转折 → ∕
    • 因果 → ∵ ∴
    • 对比 → ∥
    • 类比 → ≈
    • 下定义 → :
    • 最高级 → °
    • 分类 → 分支或数字

    笔记过程的两条附加规则

    • 分段记录：1st、2nd、3rd、4th…
    • 中英文结合 + 逐行缩进

    推荐笔记格式

    • 由上至下
    • 分段笔记
    • 逐行缩进（树状结构）

    对笔记的评价标准

    • 拥有清晰的逻辑脉络
    • 看笔记能有效复述整篇文章（过程和线索完整）

    阅读文章练习的5大处理标准（精读必做）

    1. 推测生词含义并查找归类
    2. 对理解有障碍的句子分析长难句语法
    3. 对主体段落进行 TS + D（主题句+发展方式）结构分析
    4. 做出全文结构路线图
    5. 认真分析错题并总结思路
