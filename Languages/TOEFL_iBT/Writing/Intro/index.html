    介绍

    托福写作 评分标准

    • 题量与时间：2 个任务，总计约 29–30 分钟
      • Task 1：Integrated Writing（综合写作）— 20 分钟
      • Task 2：Writing for an Academic Discussion（学术讨论帖）— 10 分钟
    • 计分方式：
      • 每个任务由评分员按 0–5 分 打分（结合机器评分）。
      • 两题平均后换算成 0–30 的写作标准分。

    • 主要评分维度：

      1. Task Achievement / Task Fulfillment：是否完成任务要求、抓住关键信息/观点。
      2. Organization & Coherence：结构是否清晰，有无段落和连接词，逻辑是否顺畅。
      3. Language Use：语法、词汇多样性与准确度。
      4. Development：内容是否有足够细节、解释和例子。

    • 能力等级（官方描述简化）：

    写作分数 能力等级 粗略描述
    24–30 Advanced 文章结构清楚，论点发展充分，语言自然流利，错误少。
    17–23 High-Intermediate 能就大多数话题写出较清晰的文章，个别地方逻辑/语言略弱。
    13–16 Low-Intermediate 能写简单短文，观点和细节比较有限，错误偏多。
    7–12 Basic 只能写出非常基础的句子，组织和语法问题明显。
    0–6 Below Basic 无法用写作有效表达。

    托福写作 题型

    1. Integrated Writing 综合写作（20 分钟）

    • 流程：
      1. 读一篇学术短文（约 3 分钟，200–250 词）。
      2. 听一段讲座（约 2 分钟），通常是支持或反驳阅读中的观点。
      3. 根据听力为主、阅读为辅写一篇总结（建议 250–300 词左右）。
    • 任务要求：
      • 准确概括讲座中三个左右的主要观点/反驳点。
      • 明确说明这些观点如何对应阅读中的三个论点。
      • 不需要写个人观点，也不要“抄”阅读原句。

    2. Writing for an Academic Discussion 学术讨论帖（10 分钟）

    • 场景：类似在线课程讨论区。
      • 教授先发一个问题（通常是学术/校园相关话题）。
      • 两位同学先发了各自的回应。
      • 你需要读完问题和同学回复，然后发表自己的帖子。
    • 任务要求：
      • 至少 100 词（建议 110–150 词）。
      • 明确表达自己的立场（赞同谁或提出新观点）。
      • 用 2–3 个理由支持观点，可以引用/评论同学的内容。
      • 注意语气自然，像真的在讨论区发帖，而不是写正式议论文。

    托福写作 常见题材与角度

    • 综合写作常见学科：
      • 生物、环境、历史、考古、社会科学、艺术史等。
    • Academic Discussion 常见话题：
      • 校园政策与学习方式（线上/线下课程、小组合作、评分制度等）。
      • 社会现象（科技、环保、城市发展、教育公平等）。
      • 日常选择（兼职、社团、住宿方式、课外活动等），但以“学术讨论”的语气呈现。
