## 椅听法核心理念（一句话记住）

不记笔记 + 50秒主动分层清内存 + 把听力当阅读做 + 抓住关键词后的解释 + 主动预判出题点 = 听力28+稳定

## 根本心态与原则

1. 没有精听和泛听之分，只有听懂和听不懂之分  
2. 前提：有一定单词量，能看懂文本  
3. 听力绝对不要记笔记（一心不能二用）  
	- 听不懂的东西记了也白记，记了也全忘  
	- 真正重要的内容题目会自带提示  
	- 听懂70%就够出题了  
	- 笔记只用于口语和作文  
4. 听懂才是王道，听不懂的笔记救不了你

## 把听力当阅读做：关键词抓重点法

所有以下词一出现，立刻竖起耳朵，它后面99%会出现同义替换/解释/正确答案：

1. 转折词（最高级别的重点）  
	but, well, yet, however → 后面是重点，常出细节题或态度转变

2. 总结词  
	so → 单层总结，常接正确选项  
	天使音 → 教授自己总结或复述学生问题，常出主旨/目的题

3. 定义/指代词  
	is, means, refers to → 常成为题干

4. 学术三件套（常对比出现）  
	fact → theory → hypothesis（一般会出现2-3个）

5. 其他高频考点信号词  
	- solve the mystery（2-3道题）  
	- A与B的关系 → correlation（双对象题）  
	- 专有名词（突然冒出）  
	- 模拟发音、期待作为题干出现

## 50秒分层法（主动清脑内存的核心技巧）

托福教授喜欢每40-50秒讲一个小点，听到以下信号立刻主动清掉上一段内存，准备接新段：

- but, well, now（最常见分层标志）  
- 突然插入题外话  
- 主题题出现太多专有名词 → 难度第二，听不懂就放弃态度题  
- 最后一层通常是两种结尾：  
	① 具体问题具体分析  
	② 交给未来研究解决

目标：从主动分层 → 到最后完全不需要分层（听懂即过）

## 托福听力出题铁律（主动出题训练）

1. 严格按照顺序出题（除重听题外）  
2. 第一题永远是主旨题（对应第0层）  
	主旨题型：态度 / 目的 / 流程 / 最准总结 / 放弃  
3. 重听题永远是最后一个  
4. 题干和正确选项可以互换，流程和理题可以互换  
5. 听到任何关键词，主动在脑子里问自己：“如果这里出题，会怎么出？”

## 标准练习流程（椅听法三遍法+倍速）

1. 第一遍：盲听（不看文本，直接做题）  
2. 第二遍：看文本+听音，做题，标记没听懂的地方  
3. 第三遍：再次盲听，做题  
4. 课后：每篇lecture再听3-4遍，用1.2倍速和1.5倍速听  
5. 总量要求：216篇lecture × 4-5遍 ≈ 100小时 ≈ 60万字

## 坚持与效果

- 听不懂没关系，托福听力一定会反复解释，越讲越细  
- 别人放弃的地方，往往就是解释最详细、最容易听懂的地方  
- 坚持完成上述流程，听力会从被动听 → 主动听 → 预测出题  
- 最终效果：听力稳定28-30分

一句话总结椅听法本质：  

> 把听力完全按照阅读的逻辑来处理，主动分层、抓关键词、预判出题、绝不记笔记，最终把被动听力变成主动阅读式听力。

这套方法的核心就是把托福听力彻底“降维打击”，用阅读的熟练度去碾压听力。
