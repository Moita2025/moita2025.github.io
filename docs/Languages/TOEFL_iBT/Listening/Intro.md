## 托福听力 评分标准

- 题量与时间：约 28 题，36 分钟左右。  
- 计分方式：
	- 每道题通常 1 原始分（少量题可能多分值），原始分满分约为 28。  
	- 原始分通过转换表换算为 0–30 的听力标准分。  
- 能力等级（官方描述简化）：

| 听力分数 | 能力等级 | 粗略描述 |
| :--: | :--: | :-- |
| 22–30 | Advanced | 基本能听懂绝大多数学术讲座和校园对话，细节与隐含信息把握较好。 |
| 17–21 | High-Intermediate | 能理解主要内容和关键细节，复杂或语速较快时偶有遗漏。 |
| 9–16 | Low-Intermediate | 熟悉主题尚可听懂大意，但细节和推理理解较弱。 |
| 0–8 | Below Low-Intermediate | 仅能听懂极少量信息，理解严重受限。 |

> 实战使用时，一般只看“标准分”是否达到目标（比如 22+ 或 25+），不需要自己手算换算表。

## 托福听力 题型

**总体结构：**

3 段讲座，每段约 3–5 分钟／6 题；2 段对话，每段约 3 分钟／5 题。

**常见 8 大听力题型：**

1. **Gist – Content / Main Idea 主旨题**  
	- 问整段对话/讲座在讲什么、主要目的是什么。
2. **Gist – Purpose 目的题**  
	- 多出现于对话，问“说话者为什么来”“老师这么说是为了做什么”。
3. **Detail 细节题**  
	- 问明确陈述的事实、数字、原因、结果等，是数量最多的一类。
4. **Understanding Function 功能题**  
	- 问某句话在语境中的功能（举例、解释、反驳、提醒、建议等）。
5. **Understanding Attitude 态度题**  
	- 判断说话者的观点或态度（赞同、反对、担心、惊讶等）。
6. **Inference 推理题**  
	- 根据信息推断未明说的结果、原因或观点。
7. **Understanding Organization 结构题**  
	- 考文章结构、信息安排方式（例如“为什么先讲 A 再讲 B”）。
8. **Connecting Content 信息归纳题**  
	- 常见形式为表格题/多选题，要求把不同部分的信息对应起来。

> 解题思路可对应整理为：主旨—细节—推理—态度—功能—结构—归纳七大板块来复习。

## 托福听力 场景与题材

- **校园生活类对话**
	- 学生与老师：选课、作业、成绩、推荐信、毕业要求等。
	- 学生与职员：图书馆、宿舍、财务、注册、活动中心等。
- **学术讲座类**
	- 自然科学：生物、地质、天文、环境、化学、物理等。
	- 社会科学：心理学、社会学、经济学、教育学等。
	- 人文艺术：历史、考古、艺术史、文学、音乐史等。
