## 托福口语 评分标准

- 题量与时间：4 个任务（1 独立 + 3 综合），约 16–17 分钟。  
- 计分方式：
	- 每个任务由评分员按 **0–4 分** 打分（部分还结合机器评分）。  
	- 4 个任务的分数综合换算成 0–30 的口语标准分。  
- 评分三大维度（对应你思维导图里的内容）：
	1. **Delivery**：语音语调、流利度、语速、停顿是否自然，是否听得清楚。
	2. **Language Use**：语法、词汇范围与准确度。
	3. **Topic Development**：是否完整回答题目；观点是否清楚、有逻辑、有细节支持。
- 能力等级（官方描述简化）：

| 口语分数 | 能力等级 | 粗略描述 |
| :--: | :--: | :-- |
| 25–30 | Advanced | 表达流利，自然停顿，错误少，能清晰表达复杂观点。 |
| 20–24 | High-Intermediate | 可以较清楚地表达意见，偶有语法/用词问题但不严重影响理解。 |
| 16–19 | Low-Intermediate | 能表达基本想法，但句型简单、停顿多，细节与结构较弱。 |
| 10–15 | Basic | 只能就熟悉话题做有限表达，错误较多、难以展开。 |
| 0–9 | Below Basic | 口语表达严重受限。 |

## 托福口语 题型（4 个任务）

1. **Question 1：Independent Speaking（个人偏好题）**
	- 话题多与日常生活、学习经历、习惯和观点有关。
	- 15 秒准备时间，45 秒作答。
	- 常见形式：二选一、多选、是否同意、优缺点比较等。

2. **Question 2：Integrated Campus – Campus Announcement（校园公告题）**
	- 先阅读一段校园公告/邮件/通知（约 45–50 秒）。  
	- 再听两名学生讨论，通常一人支持/反对该变动。  
	- 30 秒准备，60 秒作答：  
		- 简述阅读中的政策/变化  
		- 概括学生观点与理由，并说明为什么。

3. **Question 3：Integrated General-to-Specific（概念 + 例子题）**
	- 先读一段学术概念短文（定义/理论/现象），再听课堂讲解举例。  
	- 30 秒准备，60 秒作答：  
		- 简要说明阅读中定义  
		- 重点概括听力中的一个或两个例子，说明如何体现该概念。

4. **Question 4：Integrated Academic Lecture（课堂讲座总结题）**
	- 只听一段学术讲座（通常介绍两种原因/特征/方式）。  
	- 20 秒准备，60 秒作答：  
		- 先总说讲座主题  
		- 再分点讲两个要点及例子，最后一句简单总结。

## 托福口语 其他实用信息

- 可以且应该记笔记：尤其是 Task 2–4。  
- 评分更看重“是否说清楚、有条理”，而不是“难词高级句”。  
- 高分答案特点：
	- 开头主题句清晰；  
	- 每个理由有 1–2 句解释 + 具体例子；  
	- 使用简单连接词：first, second, for example, as a result, because, so, however 等。
