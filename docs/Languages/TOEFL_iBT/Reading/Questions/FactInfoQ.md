## 题目识别

**内容类**

> According to the paragraph, which of the following is true of X? 
> 
> The author’s description of X mentions which of the following?

**逻辑类**

> According to the paragraph, X occurred because… 
> 
> According to the paragraph, X did Y because… 
> 
> According to the paragraph, why did X do Y?

## 方法技巧

### 虚假比较（常见陷阱）

**核心原则：** 

若选项中出现比较级、最高级、绝对化描述或范围扩大，则**通常为错误选项**，  
除非原文存在**完全对应的比较结构**。

**常见特征：**

- more / most / less / least
- only / all / any / every
- bigger / earlier / more significant 等
- 明显扩大范围的表达（e.g. “affect all species”）

### 答非所问（逻辑方向错误）

**检验方法：因为 → 所以法**

- 将选项作为“因为”，题干作为“所以”
- 看两者是否构成清晰的因果或指向关系
- 若无法对应，则该选项答非所问

**典型错误：**

- 选项提到文中信息，但与题目问点无因果关系
- 选项内容对，但**不是题目问的点**

## 定位方法

### 利用“独特词”定位

- 在选项中寻找题干没有出现的词（尤其是名词/动词），作为定位点
- 选项为长句时，应抓主语 + 谓语的句子主干进行定位
### 定位步骤

1. **扫描（Scanning）**：快速在原文中寻找关键词、同/反义词、上下义词
2. 找到词汇第一次出现的位置（通常为答案区域）
3. 根据上下文理解实际含义
4. 匹配选项，选择符合原文事实的选项
5. 若无匹配选项，重新检查：
    - 读题是否错误？
    - 定位是否偏了？
    - 是否误解原文？

## 考试趋势与难点

现代托福阅读中，**定位句往往不是答案句** → 需扩大理解范围。

**情况 1：出现代词（如 it / they / these / this 等）**

- 需回读前文找到代词指代对象
- **答案常落在代词的真实指代内容中**

**情况 2：定位句后有逻辑词**

如：however, in addition, therefore, similarly, instead, consequently

- 逻辑词表明后一句可能是补充、对比或转折信息
- 答案往往可能在**下一句**或**相关逻辑连接的句子**中

**情况 3：定位句重复题干（无信息价值）**

- 若定位句只是题干的同义复述 → 并未给出答案
- 应继续往下读，直到找到真正回答问题的句子
