## 题目识别

> According to paragraph 1, all of the following are true of stream sorting **EXCEPT**
>
> Which of the following is **NOT** mentioned in paragraph 1 as ...

## 方法技巧

对于长文本问题，应当：

- ✔️ 带着四个选项看文本，而不是
- ❌ 每带着一个选项看一遍文本

定位方法：

- 使用**其他选项和题干没有出现的词**作为定位词，
- 当选项为长语句时，同样需要抓住**选项句子的主干**

错误选项特征

- ⽆
- 误：不符合原⽂的表达
- 偏：不符合题⽬要求，偏题
