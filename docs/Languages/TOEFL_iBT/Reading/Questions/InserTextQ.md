## 题目识别

> Look at the four squares \[■] that indicate where the following sentence could be added to the passage. 
> 
> All three of them have strengths and weaknesses, but none adequately answers all of the questions the paintings present. 
> 
> Where would the sentence best fit?

## 方法技巧

### 阅读句子 → 捕捉线索 → 顺读定位 → 代入验证

#### 1. 阅读给定句子（题干句）

- 特别注意**句首、句尾**（常含关键信息或逻辑转折）

#### 2. 判断线索词

识别能帮助定位的信号：

- **代词**：it / they / this / that / these / the + 名词 → 指代前文
- **连接词**：however, therefore, in addition, because
- **重复概念**：出现题干中的关键词或其同义表达

> 代词或 the+名词 = 往前找指代对象  
> 逻辑词 = 注意前后句的逻辑关系

#### 3. 顺读原文，寻找匹配内容

- 根据题干主干与关键词
- 寻找同义词、近义词、上下义词
- 找到相似句后继续**顺读前后 1–2 句**

#### 4. 代入验证

- 将选项带回原文上下文
- 检查前后逻辑是否通顺
- 是否直接被原文支持（不推断，不扩大）

### 结合题干主干与段落结构判断顺序

#### 1. 题干主干 → 原文相似内容

- 提炼题干主语 + 谓语作为定位主干
- 在原文中找最相似的结构或意思

#### 2. 段落结构规律（高频）

许多托福段落呈现 **总–分** 结构：

1. **总述句（General）**
    - 抽象、句子偏短
    - 位置靠前（常为段首）
2. **分述句（Specific）**
    - 句子更长、包含细节
    - 位置靠后（段落展开部分）

> 若选项内容较抽象 → 往段首找  
> 若选项内容较具体 → 往段落中后部找
